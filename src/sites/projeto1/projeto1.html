O HTML5 é a nova versão do HTML4 e um dos seus principais objetivos é facilitar a manipulação dos elementos, possibilitando o desenvolvedor modificar as características dos objetos de forma não intrusiva, fazendo com que isso fique transparente para o usuário final.

O CSS (Cascading Style Sheets) é um padrão que define como os dados são apresentados no navegador. Enquanto o HTML fornece informação sobre a estrutura de um documento, o CSS indica o aspecto que ele deve ter: a fonte, o fundo, o texto, as cores dos links, as margens e a disposição dos objetos na página.

O que é a propriedade Display em CSS


O que é display em CSS? A propriedade display CSS define se um elemento é tratado como um bloco ou elemento inline e o layout usado para seus filhos, como layout de fluxo, grid ou flex. Formalmente, a propriedade display define os tipos de exibição internos e externos de um elemento.

Display Características
Block display: block significa que o elemento é exibido como um bloco, como parágrafos e cabeçalhos sempre foram. Um bloco possui espaços em branco acima e abaixo dele e não tolera elementos HTML ao lado, exceto quando ordenado de outra forma (adicionando uma declaração float a outro elemento, por exemplo).
Inline Um elemento com display: inline é exibido na mesma linha que outros elementos, ocupando apenas o espaço necessário para seu conteúdo (o que é diferente de block, que vai ocupar todo o espaço disponível, mesmo que sobre espaço após o seu conteúdo).
Inline-Block Introdução. display: inline-block trouxe uma nova maneira de criar caixas lado a lado que se fecham e se ajustam adequadamente dependendo do espaço disponível no elemento que as contém. Isso torna mais fácil criar layouts que antes eram feitos com floats. Não é mais necessário limpar floats.
Flex O display: flex funciona de uma maneira diferente dos outros displays. Quando colocamos essa propriedade em um elemento, esse elemento se torna um flex container, a partir daí podemos manipular todos os elementos filhos desse flex container com propriedades novas.
